## Algolia DocSearch

A documentação do CACiC Event Manager utiliza o [Algolia DocSearch](https://docsearch.algolia.com/) para fornecer uma busca rápida e eficiente em toda a documentação. 

Ele é a escolha oficial do Docusaurus, o que facilita a configuração e a manutenção da funcionalidade de busca.

Não utiliza-se o Typesense, pois a busca deve funcionar mesmo com o servidor principal fora do ar.
