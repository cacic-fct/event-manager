---
title: Problemas do administrador
---

| Problema                                               | Solução                                                                  |
| ------------------------------------------------------ |------------------------------------------------------------------------- |
| Não consigo acessar o painel de administração          | Verifique se você tem as permissões administrativas necessárias para acessar o painel. Se você acha que deveria ter acesso, entre em contato com a equipe de DevOps para verificar suas permissões no Keycloak. |
