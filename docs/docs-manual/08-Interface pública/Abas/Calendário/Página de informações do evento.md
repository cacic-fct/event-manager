---
title: Página de informações do evento
---

A página de informações do evento mostra os dados públicos de uma atividade específica.

Ela pode ser aberta a partir do calendário, da inscrição em grande evento, das participações do usuário ou de um link direto.

## Inscrição

Eventos avulsos podem permitir inscrição diretamente pela página.

O botão de inscrição depende de autenticação, conexão, vagas disponíveis e período de inscrição. Depois do início do evento, a inscrição não deve ficar disponível.

Eventos que pertencem a um grande evento normalmente são inscritos pelo fluxo do grande evento.

## Presença

Quando a presença on-line está aberta, a página pode oferecer acesso ao fluxo de confirmação de presença.

Se a presença já foi registrada, a página indica a confirmação.

## Informações exibidas

A página pode exibir:

- Datas e horários;
- Carga horária;
- Descrição;
- Ministrantes;
- Transmissão;
- Localização;
- Previsão do tempo;
- Link externo definido pela organização.

## Previsão do tempo

Os dados são obtidos da API [Open-Meteo](https://open-meteo.com/) usando as coordenadas geográficas do evento.

## Funcionalidade oculta

É possível clicar no emoji para copiar o ID do evento. Isso é útil para caso o usuário esteja em tela cheia no PWA e precise do ID do evento.
