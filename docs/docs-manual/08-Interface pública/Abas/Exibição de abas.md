---
title: Exibição de abas
---

A exibição das abas principais da interface pública é controlada por Feature Flags.

Isso permite ativar ou ocultar áreas da interface sem novo deploy, especialmente quando um recurso ainda está em implantação ou quando uma seção não é necessária para determinado período.

A configuração é feita pela instância [Unleash do CACiC](https://unleash.cacic.dev.br).

Não documente manualmente uma lista fixa de abas como regra de produto. A lista visível pode mudar conforme autenticação, flags e contexto do usuário.
