---
title: Confirmar presença
---

A confirmação de presença on-line permite que o próprio participante registre presença usando um código fornecido pela organização.

Esse fluxo existe apenas para eventos configurados com presença on-line e dentro do período permitido.

## Presenças pendentes

Quando há eventos disponíveis para confirmação, a página de presenças pendentes mostra as atividades em que o usuário pode registrar presença.

Se não houver eventos disponíveis, não há presença on-line aberta para aquele usuário naquele momento.

## Código de presença

O código possui 4 caracteres.

Ele pode ser digitado manualmente ou lido por QR Code quando a organização disponibilizar o código nesse formato.

O código deve ser divulgado somente no momento adequado da atividade. Divulgar cedo demais aumenta a chance de registro indevido.

## Regras

A confirmação depende de autenticação e conexão.
