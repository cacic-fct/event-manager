---
title: Informações do organizador
---

A página de informações do organizador é restrita a ministrantes e pessoas autorizadas para aquele evento, grupo ou grande evento.

Ela evita conceder acesso ao painel administrativo para tarefas simples de apoio ao ministrante.

## Eventos

A página lista os eventos relacionados ao alvo selecionado, com quantidade de inscritos e presenças coletadas.

## Código de presença on-line

Quando o evento possui presença on-line, a página exibe o código e permite baixar um código de barras para uso na atividade.

O código deve ser compartilhado apenas em momento adequado para coleta de presença.

## Lista de inscritos

Quando autorizado no cadastro do evento, o ministrante pode baixar a lista de inscritos.

Leia [Fornecimento de dados para ministrantes](../../04-Funcionamento%20do%20app/Fornecimento%20de%20dados%20para%20ministrantes.md).
