---
title: Validar certificado
---

A página de validação permite verificar se um certificado foi emitido pelo sistema.

Ela pode ser usada por participantes, organizadores e terceiros que receberam um certificado.

## Código do certificado

A validação pode ser feita informando o código do certificado ou lendo o QR Code presente no documento.

Quando o código é válido, a página mostra o nome da pessoa, documento mascarado quando disponível, alvo do certificado e atividades vinculadas.

## Download

Após validar um certificado, é possível baixar o arquivo novamente.

O download pode ser temporariamente bloqueado por excesso de tentativas. Aguarde alguns instantes antes de tentar novamente.

## Certificado não encontrado

Quando o código não é encontrado, confira se ele foi copiado corretamente.

Se o certificado deveria existir, o participante deve verificar na página de participações ou entrar em contato com a organização do evento.
