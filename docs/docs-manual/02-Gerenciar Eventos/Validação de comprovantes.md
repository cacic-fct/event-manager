# Validação de comprovantes

:::danger[Perigo]

**Os comprovantes não comprovam nada!**  
Não saia validando comprovantes só porque há uma imagem dele.

:::

A exibição de comprovante é apenas para facilitar que o organizador do evento localize o respectivo pagamento no extrato bancário da entidade responsável pela recepção do pagamento.

Todo pagamento deve ser validado somente após consulta ao extrato.
