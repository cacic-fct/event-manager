---
title: Locais
---

A tela de locais gerencia presets usados para preencher eventos com mais rapidez.

Um preset contém nome, coordenadas e descrição exibida ao público.

## Quando usar

Use presets para locais recorrentes, como salas, laboratórios, auditórios e espaços externos usados em vários eventos.

O preset ajuda a manter consistência, mas o evento armazena os dados aplicados no momento do cadastro.

## Coordenadas

Preencha latitude e longitude quando o evento deve exibir mapa.

Depois de preencher as coordenadas, confira a prévia antes de salvar. Coordenadas incorretas podem levar participantes ao prédio ou sala errada.

## Correções

Editar um preset não deve ser tratado como correção retroativa de eventos já cadastrados.

Eventos existentes mantêm os dados de local gravados neles. Para corrigir um evento divulgado, edite o próprio evento.

## Unificação

Use a unificação quando houver presets duplicados.

Na unificação, escolha qual cadastro será mantido e quais campos serão aproveitados do cadastro removido.

Eventos existentes continuam com os dados de local já gravados. A unificação organiza os presets futuros, mas não reescreve automaticamente a programação passada.
