A aba de preferências reúne configurações privadas do usuário autenticado no painel administrativo.

## Calendário administrativo

O feed administrativo de calendário é privado e deve ser tratado como credencial. Ele permite acompanhar eventos vinculados às permissões administrativas do usuário sem abrir o painel.

O link só é exibido logo depois de habilitar ou rotacionar a chave. Depois disso, o sistema mostra o estado do feed, mas não revela a chave novamente.

O feed pode ser desativado automaticamente quando o usuário perde acesso aos alvos administrativos ou fica muito tempo sem uma sessão válida. Nesses casos, gere uma nova chave somente depois de conferir se o acesso administrativo ainda faz sentido.

Super administradores possuem um feed compartilhado próprio. Rotacionar esse feed invalida o link para todos que o utilizam.

## Rotação de chave

Rotacione a chave quando o link for exposto indevidamente, compartilhado com a pessoa errada ou cadastrado em um dispositivo que não deve mais receber atualizações.

A rotação invalida o link anterior. Clientes de calendário podem manter um cache permanentemente.
