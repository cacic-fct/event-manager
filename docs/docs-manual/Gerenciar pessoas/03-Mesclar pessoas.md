# Mesclar pessoas

O administrador com permissões de `merge-candidate` pode acessar usar o recurso de mesclar pessoas para unificar registros duplicados de pessoas no sistema.

Pessoas duplicadas podem surgir a partir de importação de dados, erros de digitação ou variações no nome.

O recurso de mesclar pessoas permite que o administrador selecione um registro principal e unifique os dados dos registros duplicados, mantendo as informações relevantes e eliminando os registros redundantes. A pessoa que permanecerá herdará todas as informações dos registro mesclados, como presenças, inscrições, etc.

Não é possível indicar registros duplicados manualmente: o sistema faz a análise automática por meio de cruzamento de dados, como nome completo, e-mail e CPF, para identificar possíveis duplicatas.
