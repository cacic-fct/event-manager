# Perfil de ministrante

## Foto do perfil

Não permitimos que os usuários façam upload de fotos, para evitar problemas de direitos autorais, exibição de conteúdo impróprio e para economizar armazenamento.

Não permitimos que usuários apontem links diretos para fotos, para evitar ataques XSS, SSRF, tracking e privacy leaks e a exibição de conteúdo impróprio.

Dessa forma, usamos a imagem do avatar do Google do usuário.
