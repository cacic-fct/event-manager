# Pessoas x Usuários

No CACiC Event Manager, existe uma distinção entre "pessoas" e "usuários".

- **Pessoas**: São os participantes e palestrantes dos eventos. Não necessariamente uma pessoa necessita ter uma conta no sistema.
- **Usuários**: São as pessoas que possuem cadastro no sistema. Obrigatoriamente, um usuário será vinculado a uma pessoa, mas nem toda pessoa precisa ser um usuário.

A maioria das operações no sistema são realizadas por meio do registro de "Pessoas" e nem sempre diretamente pelo registro do usuário.

Essa distinção habilita a importação de presenças e inscrições sem que a operação dependa da existência de contas de usuário para cada participante.  
Isso é útil para eventos antigos ou que foram realizados com listas de presença em papel.
