# Esquema

O esquema é definido pelo Prisma, que é a ferramenta de ORM (Object-Relational Mapping) utilizada no backend.

Ele é responsável por mapear as tabelas do banco de dados para objetos no código, facilitando a manipulação dos dados e garantindo a integridade do banco.

Por conta disso, alterações no esquema do banco de dados não podem ser feitas manualmente, mas sim por meio da modificação dos arquivos de esquema do Prisma em `apps/backend/prisma/schema` e por meio de migrações.
